Use bots to test the app steps: 1. add bots to the app (see below) 2. in the session config set use_browser_bots=True 3. run the server (devserver) and create a session, as you open the page pages will auto-play with the browser bots 4. continue this at https://otree.readthedocs.io/en/latest/bots.html, as well as the tutorial at https://otree.readthedocs.io/en/latest/tutorial/intro.html
